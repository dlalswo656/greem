# GREEM - 패션 쇼핑몰

## 기술 스택
- **Backend** : Java 17, Spring Boot 3.2.5, Spring Security, JPA, MySQL
- **Frontend** : React 18, Vite, Axios, React Router

## 주요 기능
- 회원가입 / 로그인 (JWT)
- 상품 목록 무한 스크롤, 카테고리 필터, 검색
- 상품 상세 (사이즈/색상 옵션 선택)
- 장바구니 (체크박스 선택 주문)
- 주문/결제, 주문 내역, 영수증
- 리뷰/별점, 문의/답변
- 찜하기, 쿠폰 할인
- 관리자 페이지 (카테고리/상품/주문 관리)
